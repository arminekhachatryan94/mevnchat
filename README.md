# mevnchat
## Real-time chat using MEVN stack
## MongoDB, Express, Vue, Node, and Socket.io

#### Run Mongo Server
``` bash
sudo mongod --dbpath /data/db/ --port 27017
```

#### Back-end: Run Node Server
``` bash
cd backend
node index.js
```

#### Front-end: Run Project
``` bash
cd frontend
npm run dev
```

#### Project is running on http://localhost:8080/#/
